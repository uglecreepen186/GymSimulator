# 🏋️‍♂️ GymSimulator — 3D-симулятор тренажёрного зала

**GymSimulator** — это компьютерная игра, разработанная в рамках дипломной работы на тему *«Разработка симулятора спортзала 2D/3D на движке Godot»*.

Вы — персонаж, который попадает в тренажёрный зал. Здесь можно выполнять три различных упражнения (мини-игры), отслеживать свою статистику и улучшать результаты. Игра демонстрирует возможности Godot Engine для создания 3D-игр с нестандартной механикой взаимодействия через луч (RayCast), системой сохранения и анимациями.

---

## 🎮 Основные возможности

- **3D-мир с low-poly графикой** — комната главного героя и тренажёрный зал.
- **Управление персонажем** — движение WASD / стрелки, взаимодействие через луч.
- **Три мини-игры**:
  - *На быстроту* — нажимайте пробел как можно больше раз за отведённое время.
  - *На точность* — повторяйте последовательность клавиш.
  - *Перетаскивание коврика* — поднимите коврик и положите его в выделенную зону.
- **Автоматическое сохранение** — лучшие результаты (время и процент успеха) сохраняются в JSON-файл.
- **Таблица статистики** — отображается в комнате главного героя.
- **Интуитивный интерфейс** — главное меню, экран управления, меню паузы.

---

## 🛠️ Технологии

- **Godot Engine 4.6.2** — игровой движок.
- **GDScript** — язык программирования.
- **Blender** — создание 3D-моделей (low-poly).
- **Формат данных** — JSON для сохранения прогресса.

## 📦 Установка и запуск

### Вариант 1: Готовый исполняемый файл
1. Скачайте архив с последним релизом из раздела [Releases]() (или скачайте ZIP-архив с кодом).
2. Распакуйте архив.
3. Запустите `GymSimulator.exe`.

### Вариант 2: Открыть проект в Godot
1. Установите Godot Engine 4.6.2 (или выше).
2. Клонируйте репозиторий:  
   `git clone https://github.com/ваш_логин/GymSimulator.git`
3. Откройте проект в Godot, выбрав файл `project.godot`.
4. Нажмите клавишу F5 для запуска.
